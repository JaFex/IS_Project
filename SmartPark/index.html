Restfull urls

Details Receive data Data needed Url Url example
Used for receiving the name of all parks Return Park with Only "ID" field Does not receive data http://{server}:{port}/api/parks
Used for receiving a list of spots from given park on a given moment Return Park with list of spots Receive park ID and date on format (dd-mm-yyyy h_mm_ss PM) http://{server}:{port}/api/parks/{park.id}/date/{date.onFormat(dd-mm-yyyy h_mm_ss PM)}
Used for receiving a list of spots from given park on a given period of time Return Park with list of spots Receive park ID, inicial date and final date on format (dd-mm-yyyy h_mm_ss PM) http://{server}:{port}/api/parks/{park.id}/date/initial/{date.onFormat(dd-mm-yyyy h_mm_ss PM)}/final/{date.onFormat(dd-mm-yyyy h_mm_ss PM)}
Used for receiving a list of spots from given park on a given moment and on given status Return Park with list of spots Receive park ID, spot state, inicial date and final date on format (dd-mm-yyyy h_mm_ss PM) http://{server}:{port}/api/parks/{park.id}/date/{date.onFormat(dd-mm-yyyy h_mm_ss PM)}/{spot.status}
Used for receiving a list of spots from given park Return Park with list of spots Receive park ID http://{server}:{port}/api/parks/{park.id}/spots
Used for receiving park details Return Park Receive park ID http://{server}:{port}/api/parks/{park.id}/details
Used for receiving a spot from given spot id on a given moment Return list of spots Receive spot ID and date on format (dd-mm-yyyy h_mm_ss PM) http://{server}:{port}/api/spots/{spot.id}/details/date/{date.onFormat(dd-mm-yyyy h_mm_ss PM)}
Used for receiving a list of spots from given battery status Return list of spots Receive battery status Enum('LOW', 'GOOD') http://{server}:{port}/api/spots/sensors/{battery_status.Enum('LOW', 'GOOD')}
Used for receiving a list of spots on given park from given battery status Return Park with list of spots Receive park ID and battery status Enum('LOW', 'GOOD') http://{server}:{port}/api/parks/{park.id}/sensors/{battery_status.Enum('LOW', 'GOOD')}
Used for receiving a list of spots from given status and park Return Park with list of spots Receive park ID and status Enum('FREE', 'OCCUPIED') http://{server}:{port}/api/parks/{park.id}/rate/{status.Enum('FREE', 'OCCUPIED')}
